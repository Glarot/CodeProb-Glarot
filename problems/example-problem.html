Sum of Array Elements

Easy arrays, loops, basic-math

Problem Description

Write a function that takes an array of integers and returns the sum of all elements in the array.

The function should handle empty arrays by returning 0.

Examples

Example 1:

Input: [1, 2, 3, 4, 5]
Output: 15
Explanation: 1 + 2 + 3 + 4 + 5 = 15

Example 2:

Input: []
Output: 0
Explanation: Empty array returns 0

Example 3:

Input: [-1, -2, -3]
Output: -6
Explanation: (-1) + (-2) + (-3) = -6

Hints

  • Consider using a loop to iterate through the array
  • Initialize a sum variable to 0
  • Don't forget to handle the empty array case
  • Think about what happens with negative numbers
