Arrays Fundamentals

Fundamentals Beginner

Overview

Arrays are one of the most fundamental data structures in programming. They provide a way to store multiple values of the same type in a single variable, with each value accessible by its position (index) in the array.

Detailed Explanation

What is an Array?

An array is a collection of elements stored in contiguous memory locations. Each element can be accessed using an index, which typically starts from 0 in most programming languages.

Key Characteristics

  • Fixed Size: In many languages, arrays have a fixed size determined at creation
  • Homogeneous: All elements are of the same data type
  • Indexed Access: Elements are accessed using numerical indices
  • Contiguous Memory: Elements are stored in adjacent memory locations

Common Operations

  • Access: Retrieve an element at a specific index - O(1)
  • Update: Modify an element at a specific index - O(1)
  • Traverse: Visit all elements in sequence - O(n)
  • Search: Find an element's position - O(n) for unsorted arrays

Code Examples

JavaScript

// Creating arrays
let numbers = [1, 2, 3, 4, 5];
let fruits = ['apple', 'banana', 'orange'];
let empty = [];

// Accessing elements
console.log(numbers[0]); // 1
console.log(fruits[1]);  // 'banana'

// Updating elements
numbers[2] = 10;
console.log(numbers); // [1, 2, 10, 4, 5]

// Array length
console.log(numbers.length); // 5

Python

# Creating lists (Python's version of arrays)
numbers = [1, 2, 3, 4, 5]
fruits = ['apple', 'banana', 'orange']
empty = []

# Accessing elements
print(numbers[0])  # 1
print(fruits[1])   # 'banana'

# Updating elements
numbers[2] = 10
print(numbers)     # [1, 2, 10, 4, 5]

# List length
print(len(numbers))  # 5

Common Patterns

// Iterating through an array
for (let i = 0; i < numbers.length; i++) {
    console.log(numbers[i]);
}

// Using forEach (JavaScript)
numbers.forEach(num => console.log(num));

// Finding sum of array elements
let sum = 0;
for (let num of numbers) {
    sum += num;
}
console.log(sum);
